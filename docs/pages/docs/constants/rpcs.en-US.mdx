# RPC Providers

## alchemyRpcUrls

RPC URLs supported by [Alchemy](https://alchemy.com/?a=wagmi).

```ts
import { alchemyRpcUrls } from 'wagmi'

// alchemyRpcUrls.mainnet
// alchemyRpcUrls.arbitrum
// alchemyRpcUrls.optimism
// alchemyRpcUrls.polygon
```
